# UnityRunner_HWA1

Васильева Александра-Ксения

Описание:
3D раннер, где игрок может двигаться в стороны и прыгать и должен уклоняться от рандомно появляющихся препятствий. Имеется 3 вида препятствий с 2 различными видами характеристик (урон и скорость).  

Главная сцена:
- SampleScene

Основные скрипты:
- PlayerController
- ObstacleSpawner

Дополнительные функции:
- Новая система ввода

Материалы:
- Множество запросов в поисковик "Unity, как сделать ___"
